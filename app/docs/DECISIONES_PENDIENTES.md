# Decisiones necesarias antes de activar registro y datos reales

## Autenticación

Rutacafetal necesita un número de teléfono como canal de contacto. Falta elegir el método para confirmar identidad y permitir que cada persona ingrese a su cuenta:

1. Teléfono con código OTP por SMS o WhatsApp: mejor experiencia para el público objetivo, pero requiere proveedor y presupuesto.
2. Correo electrónico + teléfono obligatorio: más económico para empezar, pero menos accesible para algunas personas.
3. Piloto manual: el administrador crea o aprueba altas antes de abrir el registro público.

No se debe implementar el formulario real hasta elegir una alternativa y definir recuperación de acceso.

## Política operativa inicial

- Las campañas nuevas pasan por revisión manual antes de publicarse.
- Los números de teléfono no son públicos.
- El botón de WhatsApp se habilita únicamente al patrón que aceptó una postulación.
- Las calificaciones requieren una asignación completada.
- Los reportes son privados y se moderan antes de cualquier acción o publicación.
